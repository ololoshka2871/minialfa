## Connections

### Display
SPI:
MOSI - GPIO23
MISO - NC - -
SCK - GPIO18
SS - GPIO5

GPIO:
RESET - GPIO22
DC - GPIO21

### Sensors
I2C:
SDA - GPIO26
SCL - GPIO25

### Encoder
GPIO:
BUTTON - GPIO19
V1 - GPIO16
V2 - GPIO4

### JTAG debugging
TDO - GPIO15
TMS - GPIO14
TCK - GPIO13
TDI - GPIO12
TRST - EN
VCC - +3.3V
GND - GND

### UART
RX - GPIO3
TX - GPIO1
